⚡ Connexion Firebase

Pour synchroniser vos données sur tous vos appareils, collez votre configuration Firebase ci-dessous.

  1. Ouvrez console.firebase.google.com
  2. Votre projet → ⚙️ Paramètres → Vos applications → icône Web </>
  3. Copiez le contenu de firebaseConfig = { … }
  4. Collez-le ici et cliquez Connecter
